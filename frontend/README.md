# TechCorp 企业官网 - 前端

基于 Next.js 16 + Tailwind CSS 构建的现代化企业展示网站。

## 技术栈

- Next.js 16
- Tailwind CSS
- TypeScript

## 开发

```bash
npm install
npm run dev
```

访问 http://localhost:3000

## 构建

```bash
npm run build
```

## 项目结构

```
src/
├── app/                # 页面路由
│   ├── about/          # 关于我们
│   ├── contact/        # 联系我们
│   ├── news/           # 新闻动态
│   ├── services/       # 产品服务
│   ├── solutions/      # 解决方案
│   └── page.tsx        # 首页
├── components/
│   ├── home/           # 首页组件
│   ├── layout/         # 布局组件
│   └── ui/             # 通用UI组件
```
